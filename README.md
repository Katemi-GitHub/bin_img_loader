 # bin_img_loader
 files are structured in binary like this:  
 X_size Y_size type(0 = 4-bit colors, 1 = 5-bit colors, 2 = 6-bit colors) pixel_size  
 pixel1_color pixel2_color pixel3_color ...  
 next row of pixels ...
